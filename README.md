# GoQuizBot – Telegram-викторина по языку Go

**Модуль:** Go60/quiz_bot  
**Автор:** Гоша (участник Go365)  
**Дата:** 1 марта 2026 года  

## Описание

Интерактивный бот для проверки знаний Go. Задаёт вопросы с четырьмя вариантами ответов, начисляет EXP за правильные ответы, ведёт статистику и таблицу лидеров.

## Команды

- `/quiz` – получить случайный вопрос
- `/score` – показать свою статистику (уровень, EXP, количество ответов)
- `/leaderboard` – топ-10 игроков
- `/help` – справка

## Геймификация

- За каждый правильный ответ начисляется EXP (вопросы разной сложности).
- Уровень = floor(EXP/100) + 1.
- Статистика сохраняется в файл `users.json` и восстанавливается после перезапуска.

## Установка и запуск

1. Получите токен у [@BotFather](https://t.me/botfather).
2. Вставьте токен в `main.go` (константа `BotToken`).
3. При необходимости отредактируйте `questions.json` (добавьте свои вопросы).
4. Установите зависимости и запустите:

```bash
cd Go60/quiz_bot
go mod tidy
go run main.go
